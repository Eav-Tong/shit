## 1. 产品概述

### 1.1 产品定位

一款面向考研人群的 **陪伴式学习助手** 移动端Web应用，通过虚拟学宠形象为考生提供情感陪伴与学习辅助，兼顾学习提效与情绪疏导两大核心需求。

### 1.2 目标用户

| 用户群体 | 特征描述 |
|----------|----------|
| 在校考生 | 全职备考，备考周期6-12个月 |
| 在职考生 | 边工作边备考，时间碎片化 |
| 二战/三战考生 | 有备考经验，承受较大心理压力 |

### 1.3 核心痛点

| 痛点 | 描述 | 解决方案 |
|------|------|----------|
| **备考孤独感强** | 独自学习，缺乏陪伴与监督 | 学宠实时陪伴，模拟养成互动 |
| **情绪压力大** | 焦虑、紧张、自我怀疑 | 情绪识别 + 认知调节话术 |
| **自制力不足** | 容易分心、拖延 | 番茄钟 + 打卡机制 + 决策盒子 |
| **选择困难** | 科目学习顺序纠结 | 随机决策盒子帮助快速决策 |
| **情绪无处倾诉** | 心事无处诉说，积压负面情绪 | 情绪树洞 + 智能情绪识别 |

### 1.4 核心目标

1. **降低备考孤独感** - 通过学宠陪伴提供情感支持
2. **提升学习专注度** - 通过番茄钟与打卡机制
3. **缓解负面情绪** - 通过情绪识别与智能引导
4. **增强学习动力** - 通过养成系统与成就反馈

### 2.2 P0 核心功能详情

#### C01 定制番茄钟

**功能描述**：按考研科目预设学习时长，专注计时，结束后触发学宠互动奖励

**功能需求**：
| 需求点 | 详细说明 |
|--------|----------|
| 科目预设 | 政治25min、英语30min、数学45min、专业课30min、复盘15min |
| 计时显示 | 圆形进度环 + 倒计时数字 + 当前科目 |
| 开始/暂停 | 支持暂停、继续操作 |
| 结束提醒 | 结束前1分钟轻声提醒（学宠消息推送） |
| 完成奖励 | 完成后 +10成长值，学宠发送鼓励语 |

**交互路径**：首页点击"定制番茄钟" → 选择科目 → 开始计时 → 计时结束获得奖励

---

#### C02 动态学宠陪伴

**功能描述**：虚拟学宠形象陪伴，支持两种风格切换，识别负面情绪推送话术

**功能需求**：
| 需求点 | 详细说明 |
|--------|----------|
| 学宠形象 | 猫咪形象，带名字和等级显示 |
| 风格切换 | 温柔鼓励 / 毒舌鞭策 两种模式 |
| 随机鼓励 | 点击获取随机鼓励语 |
| 情绪识别 | 识别负面情绪关键词，自动推送调节话术 |

**交互路径**：首页点击"学宠陪伴" → 选择风格 → 点击获取鼓励 / 或在负面情绪时自动触发

---

#### C03 随机决策盒子

**功能描述**：用户输入纠结的选项，随机抽取帮助快速决策，附带互动彩蛋

**功能需求**：
| 需求点 | 详细说明 |
|--------|----------|
| 选项输入 | 多行文本输入，每行一个选项 |
| 随机抽取 | 随机选择一个选项作为结果 |
| 结果展示 | 放大显示选中项，附带鼓励话术 |
| 成长奖励 | 每次使用 +2成长值 |

**交互路径**：首页点击"决策盒子" → 输入选项 → 点击开启 → 查看结果

---

#### C04 情绪树洞记录

**功能描述**：支持文字输入记录情绪，自动识别正负情绪并推送调节内容

**功能需求**：
| 需求点 | 详细说明 |
|--------|----------|
| 文字输入 | 多行文本框，支持长文本 |
| 情绪识别 | 识别正面/负面/中性关键词 |
| 结果反馈 | 根据情绪类型推送对应调节内容 |
| 一键清空 | 支持清空所有情绪记录 |

**交互路径**：首页点击"情绪树洞" → 输入心情 → 记录情绪 → 查看反馈

---

#### C05 考研每日打卡

**功能描述**：绑定个人学习计划，打卡积累奖励同步学宠成长

**功能需求**：
| 需求点 | 详细说明 |
|--------|----------|
| 计划输入 | 输入今日学习计划 |
| 打卡操作 | 每日一次打卡机会 |
| 日历展示 | 当月日历显示打卡日期 |
| 连续天数 | 计算并显示连续学习天数 |
| 奖励机制 | 打卡 +5成长值 |

**交互路径**：首页点击"每日打卡" → 输入计划 → 点击打卡 → 查看日历

---

#### C06 轻量情绪引导

**功能描述**：内置认知调节话术库，负面情绪触发时自动推送缓解焦虑

**功能需求**：
| 需求点 | 详细说明 |
|--------|----------|
| 认知话术 | 内置缓解焦虑的认知调节内容 |
| 自我鼓励 | 点击展示鼓励短语 |
| 自动触发 | 负面情绪时在首页弹窗引导 |

**交互路径**：首页点击"情绪引导" → 查看认知调节话术 / 点击短语自我鼓励

## 3. 数据模型

### 3.1 本地存储结构

```javascript
// localStorage key: 'studyPetData'

{
  // 用户设置
  petStyle: 'gentle' | 'strict',  // 学宠风格
  petName: '小考研',               // 学宠名称
  petLevel: 5,                    // 学宠等级

  // 统计数据
  totalDays: 0,         // 累计学习天数
  totalPomodoros: 0,   // 累计番茄钟数
  totalCheckins: 0,    // 累计打卡次数
  growthValue: 0,     // 成长值

  // 业务数据
  checkins: [          // 打卡记录
    '2026-04-10',
    '2026-04-11',
    '2026-04-12'
  ],
  moods: [             // 情绪记录
    {
      text: '今天学习效率好高！',
      mood: 'positive',
      time: '2026-04-12T10:30:00.000Z'
    },
    {
      text: '模拟考砸了，好焦虑',
      mood: 'negative',
      time: '2026-04-12T14:20:00.000Z'
    }
  ]
}
```

### 3.2 数据字典

| 字段 | 类型 | 说明 | 规则 |
|------|------|------|------|
| petStyle | String | 学宠陪伴风格 | gentle-温柔 / strict-毒舌 |
| petName | String | 学宠名称 | 默认"小考研" |
| petLevel | Number | 学宠等级 | 根据成长值计算 |
| totalDays | Number | 累计学习天数 | 连续打卡天数 |
| totalPomodoros | Number | 番茄钟完成次数 | 每次完成+1 |
| totalCheckins | Number | 打卡总次数 | 每次打卡+1 |
| growthValue | Number | 成长值 | 用于解锁功能 |
| checkins | Array | 打卡日期列表 | ISO日期格式 |
| moods | Array | 情绪记录列表 | 包含文本、情绪类型、时间 |

### 3.3 成长值计算规则

| 行为 | 成长值奖励 |
|------|------------|
| 完成一个番茄钟 | +10 |
| 每日打卡 | +5 |
| 记录情绪 | +3 |
| 使用决策盒子 | +2 |

### 3.4 学宠等级规则

| 等级 | 所需成长值 | 称号 |
|------|------------|------|
| Lv.1 | 0-49 | 陪伴喵 |
| Lv.2 | 50-149 | 进阶喵 |
| Lv.3 | 150-299 | 学霸喵 |
| Lv.4 | 300-499 | 考研喵 |
| Lv.5 | 500+ | 上岸喵 |

## 4. 成功标准

### 4.1 功能验收标准

| 功能 | 验收标准 | 优先级 |
|------|----------|--------|
| 番茄钟 | 可选择科目、开始/暂停、倒计时结束有提醒、完成有奖励 | P0 |
| 学宠陪伴 | 可切换风格、点击获取鼓励语 | P0 |
| 决策盒子 | 输入多选项、可随机抽取结果 | P0 |
| 情绪树洞 | 可输入文字、识别情绪类型、显示反馈 | P0 |
| 每日打卡 | 可输入计划、打卡成功、显示日历 | P0 |
| 情绪引导 | 显示认知调节话术、点击短语有反馈 | P0 |

### 4.2 性能指标

| 指标 | 目标值 |
|------|--------|
| 首屏加载时间 | < 2秒 |
| 交互响应时间 | < 300ms |
| 本地存储响应 | < 100ms |
| 内存占用 | < 50MB |

### 4.3 用户体验标准

| 标准 | 说明 |
|------|------|
| 操作路径 | 核心功能操作路径不超过3步 |
| 隐私保护 | 情绪记录默认本地存储，不上传云端 |
| 碎片化适配 | 支持随时退出、进度保存 |
| 容错性 | 异常输入有友好提示 |

### 4.4 非功能需求

- **隐私要求**：所有情绪记录默认本地存储，尊重用户隐私
- **性能要求**：轻量化定位，启动快、占用小
- **交互要求**：适配碎片化使用场景，操作简单

## 6. 竞品分析

### 6.1 市场上类似产品

| 产品 | 优点 | 不足 |
|------|------|------|
| Forest | 专注力工具成熟 | 无情感陪伴 |
| 考研帮 | 资讯丰富 | 功能太杂 |
| 滴答清单 | 待办管理完善 | 无考研场景 |

### 6.2 差异化定位

**考研线上学宠** 核心差异：**情感陪伴 + 学习提效 + 轻量简单**

- 区别于工具类：强调情感连接
- 区别于社区类：保持轻量简洁
- 专注考研场景的垂直需求
